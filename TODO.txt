Reference for exceptions:

WriteSingleCoil Error: (0x05)

Error code    : 0x85  (80 hex maior)
Exception code: 01 or 02 or 03 or 04

Function Code Not Supported       =  01  ILLEGAL FUNCTION
Output Address not exists         =  02  ILLEGAL DATA ADDRESS
Output Value Not 0000 an not FF00 =  03  ILLEGAL DATA VALUE
Write Single Coil Fails           =  04  SLAVE DEVICE FAILURE


WriteSingleRegister Error: (0x06)

Error code    : 0x86        (80 hex maior)
Function Code Not Supported       =  01
Output Address not exists         =  02
Output Value Not 0000 to FFFF     =  03
Write Single Coil Fails           =  04
